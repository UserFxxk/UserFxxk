- 🏗️ Sehari-hari membangun **tools dan aplikasi sebagai developer**

- 🔭 Saat ini sedang mengerjakan **proyek-proyek kecil yang seru**

- 💬 Tanya saya tentang **Web Development, OWASP Top 10, dan Bug Bounty**
